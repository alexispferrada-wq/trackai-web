# TrackAI — Landing 🎧

Página web pública de **TrackAI**, el copiloto de mezcla en tiempo real para Serato DJ.

🌐 **En vivo:** [trackai.tusitioya.cl](https://trackai.tusitioya.cl)

## Qué es TrackAI
Un add-on flotante para **Serato DJ Pro** (macOS) que lee lo que estás tocando y te sugiere la siguiente canción — compatible en BPM y tonalidad — y **aprende tu estilo** con el tiempo. Incluye carga en Serato con un clic, gráfico de BPM en vivo, aviso de repetición y un cerebro que mejora con cada sesión (tuya y de la comunidad de DJs).

## Este repositorio
Solo contiene la **página web** (landing) — un único `index.html` estático, servido por GitHub Pages con dominio propio (`CNAME`). El código del sistema es privado.

- `index.html` — la landing (self-contained, sin dependencias externas).
- `CNAME` — dominio personalizado.
